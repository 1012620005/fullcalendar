
getEventResource
================

Gets the [Resource Object](Resource_Object) that is associated with the given event.

<div class='spec' markdown='1'>
.fullCalendar( 'getEventResource', *idOrEvent* )
</div>

`idOrEvent` can either be a raw event ID, or an [Event Object](../event_data/Event_Object).

If no resource is associated with the specified event, `null` is returned.
