
refetchResources
================

Causes the resource data to be fetched and freshly rerendered.

<div class='spec' markdown='1'>
.fullCalendar( 'refetchResources' )
</div>

If the `resources` option [was specified as an function](resources_function/),
it will be recalled.
If the `resources` option [was specified as a JSON feed](resources_json_feed/)
it will be requested again.
